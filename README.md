# Covid-19
This project demonstrates practical implementation of Python, SQL, and Power BI to analyze real-world healthcare data and extract meaningful insights.
Project Overview

This project focuses on analyzing global COVID-19 clinical trial data to understand study types, trial status, and country-wise participation using data analytics techniques.

Objectives

Analyze COVID-19 clinical trial data

Identify study type and trial status distribution

Perform country-wise analysis of clinical trials

Present insights using dashboards

Dataset

The dataset includes information related to COVID-19 clinical trials such as:

Study Type

Trial Status

Locations

Country

Sponsor Details

Raw data was cleaned to handle missing values and extract country information.

Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (MySQL)

Power BI

Project Workflow

Data cleaning and preprocessing using Python

Exporting cleaned data for database use

Analysis using SQL queries

Visualization using Power BI dashboards

Key Insights

Majority of clinical trials were interventional studies

Most trials were in completed or recruiting stages

Countries such as the USA and India showed high participation
